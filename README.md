# 📊 Data Analysis Repository (Test Project)

## 📌 Overview

This repository has been created as a test project to practice version control using Git and GitHub, as well as to structure and document work related to data analysis.

It serves as a foundational step in building a professional workflow for future data projects.

## 🎯 Objectives

* Apply basic Git and GitHub operations

* Document projects following best practices
* Organize code, notebooks, and datasets efficiently
